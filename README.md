# 🏥 Hospital Management System

A web-based Hospital Management System developed using *PHP, **MySQL, **HTML, **CSS, **JavaScript, and **Bootstrap*. This project helps hospitals manage patients, doctors, appointments, prescriptions, and billing efficiently.

## 🚀 GitHub Repository

🔗 https://github.com/karishma24-Itsme/Hospital-Management-System

## 💻 Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
- XAMPP

## ⚙️ Installation

1. Clone this repository.
2. Copy the project folder *myhms1* into the *htdocs* folder.
3. Open *phpMyAdmin*.
4. Create a new database named:

sql
myhms


5. Import the *myhms.sql* file into the *myhms* database.
6. Start *Apache* and *MySQL* using XAMPP.
7. Open your browser and run:


http://localhost/myhms1/index.php
